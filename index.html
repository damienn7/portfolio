Quelques projets

Je vais vous présenter quelques projets réalisés en formation, mais d'abord parlons un peu de la formation.

La Web@cadémie est une formation qui utilise une pédagogie par projet qui vise à rendre ses alumnis autonomes et indépendants dans la conception d'applications web et dans la recherche d'informations afin d'être préparés à rencontrer diverses problématiques en entreprise et à pouvoir y répondre.

Maintenant voici quelques projets réalisés dans le cadre de la formation

{ Framework CSS } [1 semaine]

Click to see the code on github 👆

Ce projet a servi d'introduction aux framework CSS en général et au responsive. L'objectif était de créer un mini-framework avec SASS, technologie qui permettait de compiler le CSS une fois le code écrit en SCSS. Le code était donc plus aéré et moins long à écrire grâce à cette technologie.

{ Colle Battleship en CLI } [4 heures]

Click to see the code on github 👆

Cette colle consistait à reproduire le jeu de la bataille navale en CLI (command-line interface) avec PHP. Des restrictions étaient imposée tel que l'usage de commandes pour pouvoir jouer.

{ Générateur de Sprite et de CSS } [2 semaines]

Click to see the code on github 👆

Ce projet a permit d'apprendre à utiliser la librairie GD de PHP, qui sert principalement au traitement des images. Le but du projet était de pouvoir générer un sprite (série d'images regroupée au sein d'une même image) et le CSS qui allait avec afin de pouvoir récupérer une image dans le sprite sur son site web, tout cela en ligne de commande avec options depuis le CLI. Les restrictions du projet ici concernaient les options à utiliser dans le programme (recherche d'image avec option récursive, nommage des fichiers générés, emplacement du dossier à utiliser pour la recherche,...etc.).

{ Colle d'affichage SQL en CLI } [4 heures]

Click to see the code on github 👆

Cette colle visait quant à elle à utiliser la librairie PDO pour l'accès à la base de données et à interpréter les données retournées en PHP afin de les afficher de la même façon que le fait l'éditeur MySql.
